how to use it ?

  • 1. Buatlah Daftar Tugas: Masukkan tugas baru ke dalam input dan klik tombol "Add" untuk menambahkan tugas baru ke dalam daftar.
  • 2. Tandai Sebagai Selesai: Klik pada tugas yang sudah selesai untuk menandainya sebagai selesai. Tugas yang telah selesai akan memiliki garis melintang di atasnya.
  • 3. Hapus Tugas: Untuk menghapus tugas, klik tombol "X" di sebelah kanan tugas yang ingin dihapus.
  • 4. Mode Gelap/Terang: Klik ikon mode (matahari/bulan) di sudut kanan atas untuk beralih antara mode gelap dan mode terang. Ini akan mengubah tampilan halaman menjadi mode gelap atau mode terang, sesuai dengan preferensi Anda.
  • 5. Penyimpanan Otomatis: Tugas yang Anda tambahkan, tandai sebagai selesai, atau hapus akan disimpan secara otomatis di localStorage, sehingga ketika Anda kembali ke halaman, tugas-tugas tersebut akan tetap ada.
